## 🚗 Sobre o Projeto

O sistema busca facilitar a gestão de infrações de trânsito de forma eficiente, trazendo uma forma rápida e intuitiva de cadastrar, consultar, atualizar e deletar seus dados, auxiliando tanto o agente que aplica a multa quanto o cidadão comum.
